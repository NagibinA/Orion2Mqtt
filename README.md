# Orion2Mqtt - Мост между RS485 и MQTT для ESP8266
Управление и контроль устройствами Bolid через MQTT с помощью Orion2Mqtt, подключенными к RS485

![Orion2Mqtt](/images/orion2mqtt.png)

Версия: 1.0
Платформа: ESP8266
Лицензия: MIT

## Обзор

Orion2Mqtt - это мостовое приложение, которое соединяет устройства Orion (например, С2000-БКИ) через интерфейс RS485 с MQTT брокером используя ESP8266.

## Возможности

- Преобразование RS485 в MQTT
- Автоматическое обнаружение пакетов через CRC8
- Контрольная сумма CRC8 для целостности данных
- LWT (Last Will and Testament) для мониторинга статуса
- Автоматическое переподключение к WiFi и MQTT
- Управление буфером Serial

## Аппаратные требования

- Плата на базе ESP8266 (NodeMCU, Wemos D1 Mini)
- Преобразователь RS485 в TTL (MAX485)

## Подключение

| Пин ESP8266 | Модуль RS485 | Функция |
|-------------|--------------|---------|
| D4 (GPIO2)  | RE/DE        | Управление RTS |
| D1 (GPIO5)  | RO           | Прием (RX) |
| D2 (GPIO4)  | DI           | Передача (TX) |

Примечание: пины TX/RX могут отличаться.

## Программные зависимости

Библиотеки (установить через Library Manager):
- PubSubClient от Nick O'Leary (версия 2.8+)

## Конфигурация

### Настройки WiFi

const char* ssid = "Название вашей Wi-Fi сети";

const char* password = "Пароль от вашей WiFi сети";

### Настройки MQTT брокера

byte server[] = {192, 168, 1, 50};  // IP адрес

const char* clientID = "Orion2Mqtt";

const char* mqtt_username = "ваш_логин";

const char* mqtt_password = "ваш_пароль";

### MQTT топики

Топик для отправки команд (подписка): Orion2Mqtt/byteN

Топик для ответов (публикация): Orion2Mqtt/answer

Топик статуса (LWT): Orion2Mqtt/Available

### Отправка команд

В топик Orion2Mqtt/byteN отправляйте:

Адрес;Длина;Байт0;Байт1;...

Пример (запрос типа и версии прибора по адресу 2):

2;6;0;13;0;0

### Получение ответов

Ответы приходят в топик Orion2Mqtt/answer:

2 7 0 44 245 0 5 26

## Детали протокола

Формат сообщения:

- Байт 0: Адрес устройства

- Байт 1: Длина сообщения

- Байты 2..n: Данные

- Байт n+1: CRC8

## Устранение неполадок

WiFi не подключается:
- Проверьте SSID и пароль
- Убедитесь что сеть 2.4GHz

MQTT не подключается:

- Проверьте IP и порт (1883)

- Проверьте логин и пароль

Нет данных от RS485:

- Проверьте подключение MAX485

- Проверьте пин RTS (GPIO2)

- Убедитесь что скорость 9600 бод

Мусорные данные:

- Проверьте общий провод

- Проверьте терминальные резисторы
